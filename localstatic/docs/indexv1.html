核心价值

MVP 的重点是把“生成结果”产品化：可读、可保存、可重生成、可导出、可被运营分析。

结构化行程

输出稳定 JSON 背后的页面表达是每日卡片、时间段、地点、成本和注意事项，让结果可以继续编辑。

旅行库沉淀

历史计划不是消息记录，而是用户的“我的行程库”，支持打开、重生成、导出和继续反馈。

任务健康闭环

管理员能看热门目的地、成功率、失败任务和反馈评分，把 Agent 的质量运营起来。
